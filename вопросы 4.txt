1) 
К полю структуры можно обратиться  либо через итератор (struct.pole), либо через указатель (struct* = ssilka и потом ssilka->pole). 
Инициализация структуры : struct Imya{ вставляем переменные}; 
Также можно объявлять значения по умолчанию, просто тем что ты даешь им значение int x = 0.
В структуре все данные по умолчанию могут читаться извне (public), поэтому чтобы скрыть доступ к некоторым из них, эти поля можно перенести под слово private: 
{обычное тело структуры
private:
перечисление закрытых полей}

2) 
Битовое поле - элемент структуры, для которого прописывается определённое количество памяти в битах, заниманиемое этим полем.
Битовые поля используются для экономного хранения данных , в которых отдельные биты имеют самостоятельные значения.

3) 
Union это класс, в котором может храниться переменная одного из указанного в нем типа. 
Объединение экономит память, т.к. резервирует ее под самый большой из представленных в нем типов. Например, с помощью объединения 4 байта можно считать либо за 4 чара, либо за один 32битный инт.
Сейчас union почит не используется, потому что есть более удобный и безопасный std::variant

4)
Перегрузка оператора - это то же самое, что перегрузка функции. Когда мы перегружаем оператор, мы определяем его новые версии, которые могут работать с разными типами данных. 
Чтобы перегрузить оператор, надо указать новый тип, с которым он теперь может работать, затем сам оператор и его аргументы

5) 
Передача производится по ссылке т.к. сам передаваемый объект должен изменяться. Пример работы на cout << x << y
сначала << выполнит передачу х, и потом операция выглядит как cout << y.

6) Для чего применяется перечисление с областью видимости enum class? В каких случаях используется конструкция switch-case-default и как она устроена?
Еnum class нужен в тех случаях, когда нужно показать, что некая переменная должна принимать значения только из указанного списка.  
Это позволяет объявлять много классов перечисления, члены которых имеют одинаковые имена.
Конструкция switch-case-default работает как оператор if, где case то if или if else в зависимости, если это первая или последующие написания. И default = else. Но для switch не нужно фигурных скобок и написание занимает меньше времени.
